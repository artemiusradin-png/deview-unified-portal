# Commercial Offer

## Eden's Father Internal Business Data Portal

**Prepared for:** Mr. Lam  
**Prepared by:** DeView  
**Date:** April 15, 2026

## 1. Project Overview

DeView proposes to build a secure internal web portal that consolidates Eden's Father business data into one operational platform. The goal is to replace fragmented lookup workflows across multiple systems with a single search-driven interface where authorized staff can quickly find a customer, review all related loan information, and make better operational decisions.

This proposal is structured in three pricing stages so the project can match the client’s immediate needs and budget, while still allowing room for a more advanced operational platform over time.

## 2. Pricing Options

### Option 1: Foundation Portal
**USD 4,000**

This package covers the core platform described in the PRD and is the recommended starting point if the immediate goal is to launch a reliable internal portal quickly.

Included scope:

- Secure staff login
- Role-based access control
- Unified search by name, phone number, and ID number
- Search results table for quick customer identification
- Customer 360 profile page
- Core data modules requested in the brief:
  - Apply Info
  - Partakers
  - Credit Ref
  - Document
  - Mortgage
  - Income / Expenditure (DSR)
  - Loan History
  - Partaking History
  - Approval Info
  - Repay History
  - Repay Condition
  - CRM
  - OCA / Write-off
- Read-only staff interface for safe operational usage
- Basic admin access management
- Audit logging for traceability
- Manual or scheduled daily data refresh workflow

Business benefit:

This option gives the client one central working portal and immediately reduces time lost switching between different systems. It is the most cost-efficient path to operational improvement.

### Option 2: Operations Plus
**USD 6,000**

This package includes everything in Option 1, plus extra functions that improve day-to-day team efficiency and make the system more useful for operations managers.

Additional scope:

- Advanced search filters by business unit, branch, status, and loan type
- Document preview and structured file access inside the customer profile
- Export / print-ready customer summary view for internal review
- Dashboard overview of recent searches, flagged cases, and activity
- Better admin controls for user roles and permissions
- UAT support and launch configuration for smoother deployment
- Staff onboarding session and operating handover

Business benefit:

This option is better for a live operating team that needs not only data access, but also faster review, cleaner handover, and stronger internal control. It turns the portal into a more complete operations tool rather than only a lookup system.

### Option 3: Management + AI Suite
**USD 8,000**

This package includes everything in Option 2, plus management-facing and AI-supported features for higher-value decision support.

Additional scope:

- AI-generated case summaries for faster borrower review
- AI assistant for profile-level internal analysis and question answering
- Related-party and relationship visibility enhancements
- Risk flags and exception highlighting for blacklist, repayment, or approval review
- Management reporting view for portfolio oversight
- Workflow notes / internal case comments for team coordination
- Priority post-launch optimization period after go-live

Business benefit:

This option is designed for clients who want the platform to support management decisions, not just staff lookup. It creates more value for supervisors and business owners by reducing review time, improving visibility, and making complex records easier to understand.
